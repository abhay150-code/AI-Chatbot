# Aether AI Chatbot

A premium, next-generation AI chatbot featuring intelligent conversation and high-fidelity image generation.

## Features
- **Intelligent Conversation**: Powered by Gemini 2.0 via OpenRouter.
- **Image Generation**: Powered by Flux 1.0 via Hugging Face.
- **Premium UI**: Cyber-modern dark theme with glassmorphism and smooth animations.
- **Secure by Design**: API keys are stored in your browser's local storage and never pushed to GitHub.

## Getting Started

### Prerequisites
- A modern web browser.
- An [OpenRouter](https://openrouter.ai/) API Key.
- A [Hugging Face](https://huggingface.co/) Token (optional, for image generation).

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/abhay150-code/AI-Chatbot.git
   ```
2. Open `index.html` in your browser.
3. The app will prompt you for your API keys on first use. You can also update them in the **Settings** menu.

## Technologies Used
- HTML5 & CSS3 (Glassmorphism, Grid, Flexbox)
- Vanilla JavaScript (ES6+)
- Hugging Face Inference API
- OpenRouter API
